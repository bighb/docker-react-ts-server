# 项目名称

这是一个使用 Koa.js 构建的项目。

## 安装

确保你已经安装了 Node.js 和 npm。然后在项目根目录下运行以下命令来安装依赖：

```bash
npm install
```

## 运行

首先，你需要全局安装 `nodemon`。`nodemon` 是一个工具，当源文件改变时，它会自动重启你的应用。你可以使用以下命令来安装：

```bash
npm install -g nodemon
```

然后，在项目根目录下运行以下命令来启动服务器：

```bash
nodemon app.js
```

服务器默认运行在 localhost:3000。
